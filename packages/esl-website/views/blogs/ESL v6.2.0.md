The ESL v6.2.0 has been released!

This release focuses on TS6 compatibility, smoother carousel interactions, and a few high-impact ecosystem updates.

Highlights include out-of-the-box AI skills for `@exadel/esl` (beta feature for now), the updated `esl-carousel` autoplay API with better pause handling and interaction scoping, and the new `ESLMatchHeightMixin` for layout matching.

We also fixed autoplay blocker re-checking from global events and corrected `parseNumber` handling for empty-space values in `esl-utils`.
