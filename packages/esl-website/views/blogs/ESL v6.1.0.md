ESL v6.1.0 is now available!

This release improves accessibility and flexibility across core utilities and carousel components.
We’ve replaced the legacy `no-inert` hook in `esl-carousel` with a proper `focus-policy` feature and expanded the built-in `sanitize` utility to support DOM Elements and more advanced sanitization scenarios.

The update also includes several fixes, including improved JSX children support in `esl-carousel` and corrected parsing logic for `none` time values in `esl-utils`.
